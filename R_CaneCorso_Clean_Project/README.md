# R Cane Corso Regression Project

This is a simple educational R project for linear regression.

## Run

Open PowerShell in this folder and run:

& "C:\Program Files\R\R-4.6.0\bin\Rscript.exe" .\cane_corso_regression_project.R

## Output

The script prints:
- dataset preview
- train/test split
- linear regression model summary
- actual vs predicted Cane Corso weight
- MAE and RMSE model testing
- example prediction for Marcus

It also creates:
- cane_corso_age_vs_weight.png
- cane_corso_actual_vs_predicted.png
