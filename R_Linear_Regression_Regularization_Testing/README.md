# Linear Regression, Regularization and Testing in R

This project covers:

- Regression - Problem Statement and Motivation
- Ordinary Least Squares: Method, Simulated Example, Implementation on Real Data
- RANSAC - Robust Regression Model
- Linear Regression Extensions: Polynomial Regression, Multi-Dimensional Linear Regression
- Regularization
- Model Testing

## How to run

Open PowerShell in this folder and run:

```powershell
& "C:\Program Files\R\R-4.6.0\bin\Rscript.exe" .\linear_regression_regularization_testing.R
```

## Files

- `linear_regression_regularization_testing.R` - main R script
- `PROJECT_REPORT_BG.md` - short Bulgarian report in first person
- Generated PNG charts after running the script
